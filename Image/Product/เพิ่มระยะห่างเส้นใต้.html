ถ้าคุณอยากเพิ่มระยะห่างระหว่าง "เส้นใต้" (underline) กับ "ตัวหนังสือ" ใน CSS ปกติ จะใช้ text-decoration ไม่สามารถกำหนดระยะห่างได้โดยตรง แต่มีวิธี workaround โดยใช้ box-shadow หรือ border-bottom แทน เช่น:

✅ วิธีที่แนะนำ: ใช้ border-bottom แทน text-decoration

html
<span class="custom-underline">เต้าหู้ก่อน</span>

css
.custom-underline {
  display: inline-block;
  border-bottom: 4px solid orange; /* ปรับความหนา/สีได้ */
  padding-bottom: 5px; /* ระยะห่างจากตัวอักษร */
}

หรือถ้าอยากเน้นเหมือน underline จริงๆ พร้อมระยะห่าง:

css
.custom-underline {
  text-decoration: none;
  position: relative;
}
.custom-underline::after {
  content: "";
  position: absolute;
  left: 0;
  right: 0;
  bottom: -2px; /* ระยะห่างจากตัวอักษร */
  height: 2px;
  background-color: orange;
}

หากคุณส่ง HTML/CSS ที่คุณใช้มาให้ ผมสามารถแก้ให้ตรงจุดได้เลยครับ.
